# Notion Notes Summariser
Exports university notes from Notion and generates concise per-subject summaries with an LLM.
- `export_notion.py` – pull notes from a Notion workspace
- `summaries.py` – summarise per subject via the OpenAI API
Supply your own credentials in place of the `YOUR_OPENAI_API_KEY` / `YOUR_NOTION_TOKEN` placeholders.
